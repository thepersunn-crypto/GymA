## PART 2: THE 13-STEP WEBSITE SPECIFICATION

Please process the business profile above and apply the following 13 steps rigorously to design and generate the codebase for this website.

### STEP 1: DEEP BUSINESS COMPREHENSION
Analyze the provided profile to identify key themes. Emphasize the gym's **USPs** (e.g., luxury recovery, high-end coaching) throughout the copy. Use the **Target Audience** information to write copy that addresses their specific pain points (e.g., lack of time, plateaus in progress, intimidation at basic gyms).

### STEP 2: CONVERSION-FOCUSSED WEBPAGE STRATEGY
Design the layout to guide visitors down a psychological funnel:
1. **Awareness & Awe:** Hook them instantly with high-impact, premium cinematic visuals.
2. **Desire & Identification:** Showcase real results (transformations) so they think: *"This gym can transform me."*
3. **Trust & Authority:** Present certified elite trainers and actual Google reviews.
4. **Action (Low Friction):** Prompt them to act immediately with CTA buttons leading to a personalized **WhatsApp inquiry** or **Free Trial Booking**.

### STEP 3: PREMIUM BRAND DIRECTION & VISUAL MOOD
Do NOT build a generic, cheap-looking template. Apply a **Cinematic Dark Theme**:
- **Backgrounds:** Rich charcoal blacks, deep slate grays, and midnight colors.
- **Accents:** High-energy neon/vibrant tones (e.g., hyper-red, electric neon green, or molten orange) used strategically to guide the eyes to CTAs and highlights.
- **Visual Texture:** Subtle noise overlays, gradient masks, grid alignments, and premium borders to establish depth.

### STEP 4: ADVANCED MODERN UI/UX ARCHITECTURE
Implement these modern interface design patterns:
- **Glassmorphism:** Navigation menus and cards with light-refracting semi-transparent backgrounds, subtle borders, and backdrop-blur filters (`backdrop-filter: blur(12px)`).
- **Depth & Layering:** Overlapping elements, sticky scrolling sidebars, and premium smooth shadows (`box-shadow: 0 20px 40px rgba(0, 0, 0, 0.5)`).
- **Interactive Cards:** Programs and trainers should expand, flip, or scale elegantly on hover, displaying secondary content.

### STEP 5: CURATED COLOR PALETTE & TYPOGRAPHY SYSTEM
Use one of the following premium fitness color systems (and write CSS custom properties accordingly):
* **Option A (High-Energy Luxury):** Primary Background: `#0A0A0B` | Secondary: `#121214` | Accent: `#FF2E2E` (Vibrant Crimson) | Neutral Text: `#F3F4F6`
* **Option B (Electric Cyber-Fitness):** Primary Background: `#08090A` | Secondary: `#0F1115` | Accent: `#39FF14` (Electric Neon Green) | Neutral Text: `#F8FAFC`
* **Option C (Forged Power & Grit):** Primary Background: `#0B0C10` | Secondary: `#1F2833` | Accent: `#FF7F11` (Molten Orange) | Neutral Text: `#C5C6C7`

**Typography Pairings:**
- **Headings:** Bold, wide, uppercase, high-impact athletic sans-serif fonts (e.g., `Syne`, `Orbitron`, `Montserrat`, or `Clash Display` via Google Fonts).
- **Body:** Clean, highly readable, geometric sans-serif (e.g., `Inter`, `Plus Jakarta Sans`, or `Outfit`).
- Use robust letter-spacing (`tracking-wider` or `tracking-widest`) for uppercase subheadings.

### STEP 6: WEBPAGE SITEMAP

### STEP 10: AUTOMATED LOCAL SEO INJECTION
Structure the code using the best search engine optimization practices:
- Use semantic HTML tags exclusively (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`).
- Inject meta tags for Title and Description optimized for local searches. (E.g., *"Premium Gym in [Location] | World-Class Fitness, Coaching, & Recovery Lounge"*).
- Include standard JSON-LD LocalBusiness schema script templates in the HTML header.

### STEP 11: OBSESSIVE MOBILE-FIRST DESIGN
Over 80% of gym visitors will search on mobile. Ensure:
- 100% thumb-friendly interactive targets (all buttons have at least `min-height: 48px`).
- Responsive padding adjustments so headlines never wrap awkwardly.
- No horizontal layout overflow. Navigation bar collapses into an elegant full-screen glassmorphic overlay menu on mobile.

### STEP 12: PREMIUM MICRO-ANIMATIONS & TRANSITIONS

- **Interactive Counters:** When the "Transformations" or "Reviews" sections scroll into view, count up statistics smoothly (e.g., `0` to `1,500+` members).
- **Glow transitions:** Accent buttons should pulse or glow slightly on hover using CSS filters (`filter: drop-shadow(0 0 8px var(--accent-color))`).
